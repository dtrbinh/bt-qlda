Bài tập thực hành với GIT

Hướng dẫn chi tiết bài tập

  • Bước 1: Các bạn tạo repo mới để quản lý dự án bằng Github, thêm collaborators là các thành viên trong nhóm
  • Bước 2: Mỗi thành viên tạo một branch riêng để code từng phần của mình
  • Bước 3: Push code lên và merge code lại vào nhánh chính
  • Bước 4: Deplpy bằng Github pages và gửi link vào MS Teams
  • Chú ý: Mỗi thành viên đều yêu cầu merge request và không được đẩy trực tiếp lên nhánh chính
